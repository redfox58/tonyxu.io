                          # Tech stack to build your own product

                          As a software engineer or developer, you might want to build a product end-to-end. These are the frameworks and technologies you might need. All listed items are open-source or free to use.

                          # Web Front-end

                          • Fundamental Language: HTML, JavaScript, CSS
                          • Advanced Language: ES6, TypeScript, CoffeScript
                          • JS Library/Framework: React, Vue, Angular, jQuery
                          • Template Engine: Mustache, Handlebars, Jade
                          • CSS Library/Framework:
                            • Global: Bootstrap, Foundation, Semantic
                            • China: Element, Ant Design, Amaze UI
                            • Framework-Specific: Material-UI (React), Angular Material (Angular), iView (Vue)
                          • Automation/Build: Grunt, Gulp
                          • Module Bundler: Browserify, Webpack
                          • Package Management: npm, bower, yarn
                          • Code Quality: JSLint, ESLint, TSLint
                          • Cross-platform
                            • Mobile: React Native, Ionic, MUI
                            • Desktop: Electron

                          # Mobile

                          • iOS: Objective-C, Swift
                          • Android: Java, Kotlin
                          • Cross-platform: React Native, Flutter

                          # Back-end / REST API

                          • Node.js: Express
                          • Python: Flask
                          • Java: Spring

                          # Database

                          • NoSQL: MongoDB
                          • Relational DB: MySQL
                          • Realtime: Firebase

                          # Big Data

                          • Hadoop
                          • Spark

                          # Continuous Integration and Delivery (CI/CD)

                          • Jenkins
                          • Travis CI
                          • Circle CI

                          # Monitor

                          • Icinga
                          • Zabbix
                          • Nagios
                          • Datadog

                          # Cache

                          • Redis
                          • Memcached

                          # Web Server

                          • Nginx
                          • Apache

                          # CDN

                          • Global: Cloudflare
                          • China: 百度云、阿里云、网宿

                          # Cloud

                          • Global: AWS, Azure, Google Cloud
                          • China: 阿里云、腾讯云

                          # Payment

                          • Native: Apple Pay, Google Pay
                          • World: Stripe, Paypal
                          • China: 支付宝、微信支付、Ping++
